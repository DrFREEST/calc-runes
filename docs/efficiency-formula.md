## 1. 개요

### 1.1 목적
- 룬의 DPS 기여도를 수치화하여 효율 점수 산출
- 캐릭터 스텟과 연계한 상대적 효율 비교
- 시너지 효과를 반영한 최적 조합 추천

### 1.2 기본 원칙
- 모든 가중치는 0~1 스케일로 통일
- DPS에 직접 기여하는 효과만 계산에 포함
- 시너지 보너스는 별도 계산 후 합산

## 2. 기본 공식

### 2.1 개별 룬 효율 점수

```
룬 효율 점수 = Σ(효과 점수) - Σ(결함 점수)
```

### 2.2 효과 점수 계산

```
효과 점수 = 효과값 × 효과 가중치 × 유형 가중치 × 업타임
```

| 항목 | 설명 | 예시 |
|---|---|---|
| **효과값** | 효과의 수치 (%) | 공격력 20% → 20 |
| **효과 가중치** | DPS 기여도 (0~1) | 공격력 → 1.0 |
| **유형 가중치** | 발동 유형 (0~1) | 상시 → 1.0, 트리거 → 0.8 |
| **업타임** | 실제 적용 비율 (0~1) | 지속 20초/쿨 90초 → 0.182 |

### 2.3 결함 점수 계산

```
결함 점수 = 결함값 × 결함 가중치
```

### 2.4 조합 총 효율

```
조합 총 효율 = Σ(개별 룬 효율) + Σ(시너지 보너스)
```

## 4. 유형 가중치

### 4.1 효과 유형별 가중치

| 유형 | 가중치 | 설명 |
|---|---|---|
| PERMANENT | 1.0 | 상시 적용 |
| STACKING | 0.95 | 최대 중첩 기준, 쉽게 유지 |
| TRIGGER | 0.8 | 조건 발동 |
| STATE | 0.7 | HP/자원 조건 |
| DURATION | 업타임 계산 | 지속시간/쿨타임 |
| DECAY | 특수 계산 | 평균값 적용 |

### 4.2 업타임 계산

```
업타임 = 지속시간 / (지속시간 + 쿨타임)
```

**예시:**
- 지속 20초, 쿨타임 90초 → 20 / 110 = 18.2%
- 지속 5초, 쿨타임 3초 → 5 / 8 = 62.5%

### 4.3 DECAY 효과 계산 (시간 감소 효과)

```
평균 효과값 = 초기값 × (감소완료시간 / 2) / 전투시간
```

**예시 (땅울림):**
- 초기 30%, 3초마다 2% 감소, 45초 후 0%
- 전투시간 120초 (레이드 기준)
- 평균값 = 30 × (45/2) / 120 = 5.625%

### 4.4 각성 효과 업타임

```
각성 업타임 = 각성 지속시간 / (각성 지속시간 + 각성 쿨타임)

기본 각성 쿨타임: 90초
눈 먼 예언자 장착 시: 90 - 38 = 52초
```

| 엠블럼 | 지속 | 기본 업타임 | 눈먼예언자 장착 시 |
|---|---|---|---|
| 산맥 군주 | 20초 | 18.2% | 27.8% |
| 부서진 하늘 | 20초 | 18.2% | 27.8% |
| 아득한 빛 | 20초 | 18.2% | 27.8% |
| 갈라진 땅 | 35초 | 28.0% | 40.2% |
| 흩날리는 검 | 35초 | 28.0% | 40.2% |
| 대마법사 | 20초 | 18.2% | 27.8% |

## 5. 결함 가중치

### 5.1 결함 종류별 가중치

| 결함 | 가중치 | 비고 |
|---|---|---|
| 피해량 감소 | 1.0 | 직접 DPS 감소 |
| 멀티히트 피해 감소 | 0.8 | 클래스별 영향도 상이 |
| 쿨타임 회복 속도 감소 | 0.7 | 스킬 사용 빈도 감소 |
| 치명타 확률 감소 | 0.6 | |
| 스킬 사용 속도 감소 | 0.5 | |
| 캐스팅/차지 속도 감소 | 0.5 | |
| 이동 속도 감소 | 0.1 | DPS 거의 무관 |
| 받는 피해 증가 | 0.2 | 생존 관련 |

### 5.2 결함 점수 계산 예시

**은둔자:**
- 공격력 +33% → 33 × 1.0 × 1.0 = 33점
- 쿨감 -9% → 9 × 0.7 = 6.3점
- 최종: 33 - 6.3 = **26.7점**

**투사+:**
- 피해량 +33% → 33 × 1.0 × 1.0 = 33점
- 멀티히트 -10% → 10 × 0.8 = 8점
- 최종: 33 - 8 = **25점**

## 6. 시너지 보너스

### 6.1 시너지 유형

| 시너지 유형 | 제공 룬 | 수혜 룬 | 보너스 계산 |
|---|---|---|---|
| 결함 제거 | 영원 | 결함 있는 룬 | 결함 점수 전액 복구 |
| 각성 쿨감 | 눈 먼 예언자 | 각성 엠블럼 | 업타임 증가분 반영 |
| 각성 2배 | 각성 엠블럼 | 압도적인힘/섬세한손놀림 | 추가 효과 × 업타임 |
| 중첩 보너스 | 쌍둥이별/에메랄드숲 | 중첩 룬 | 중첩 +1 효과 |
| DoT 시너지 | DoT 부여 룬 | DoT 조건 룬 | 조건 효과 발동 |

### 6.2 결함 제거 시너지

```
영원 + 결함 룬 시너지 보너스 = 결함 점수 전액

예시: 영원 + 은둔자
- 은둔자 결함: 쿨감 -9% = 6.3점
- 시너지 보너스: +6.3점
- 은둔자 최종: 33점 (결함 제거됨)
```

### 6.3 각성 쿨감 시너지

```
눈 먼 예언자 + 각성 엠블럼 시너지 보너스
= (수정 업타임 - 기본 업타임) × 각성 효과 점수

예시: 눈 먼 예언자 + 대마법사
- 기본 업타임: 18.2%
- 수정 업타임: 27.8%
- 각성 효과 (공격력 45%): 45 × 1.0 = 45점
- 시너지 보너스: (0.278 - 0.182) × 45 = 4.3점
```

### 6.4 각성 2배 시너지

```
각성 엠블럼 + 압도적인힘/섬세한손놀림 시너지 보너스
= 추가 효과 점수 × 각성 업타임

예시: 산맥 군주 + 압도적인 힘
- 압도적인 힘 기본: 강타 6% + 공격력 6% + 치확 4%
- 각성 시 추가: 강타 +6% + 공격력 +6% + 치확 +4%
- 추가 효과 점수: 6×0.5 + 6×1.0 + 4×0.8 = 12.2점
- 각성 업타임: 18.2%
- 시너지 보너스: 12.2 × 0.182 = 2.2점
```

### 6.5 중첩 보너스 시너지

```
쌍둥이별/에메랄드숲 + 중첩 룬 시너지 보너스
= 1중첩 효과값 × 효과 가중치

예시: 쌍둥이 별 + 검무+
- 검무+ 중첩: 공격력 2%/회, 최대 10회
- 쌍둥이 별: 누적 +1
- 시너지 보너스: 2 × 1.0 = 2점
```

### 6.6 DoT 시너지

```
DoT 부여 룬 + DoT 조건 룬 시너지 보너스
= 조건 효과 점수 × 트리거 가중치

예시: 야수+ (출혈 부여) + 절개+ (출혈 → 공격력 20%)
- 절개+ 효과: 공격력 20% (트리거)
- 시너지 보너스: 20 × 1.0 × 0.8 = 16점
```

### 6.7 시너지 매트릭스

| 제공 룬 | DoT 종류 | 수혜 룬 | 효과 |
|---|---|---|---|
| 야수+ | 출혈 | 절개+ | 공격력 +20% |
| 불길 | 화상 | 들불 | 치명타 확률 +14% |
| 결정 | 빙결 | 설산 | 추가타 확률 +16% |
| 뇌명 | 감전 | 세상을 삼키는 악의 | 피해량 +16% |
| 밤+ | 정신 | 환영+ | 받는 피해 +10% |
| 메아리치는 진노 | 화상/빙결 | 들불/설산 | 복합 시너지 |

## 7. 캐릭터 스텟 연계

### 7.1 스텟 입력 항목

| 구분 | 스텟 | 입력 형식 |
|---|---|---|
| 기본 5스텟 | 힘, 솜씨, 지력, 의지, 행운 | 수치 |
| 주요 스텟 | 공격력, 방어력 | 수치 |
| 공격 스텟 | 브레이크, 강타강화, 콤보강화, 스킬위력, 광역강화, 연타강화, 추가타, 치명타 | 수치 |
| 속도 스텟 | 빠른공격, 빠른스킬, 궁극기 | 수치 |
| 방어 스텟 | 피해감소, 급소회피, 회복력, 추가체력 | 수치 |

### 7.2 스텟 수치 효과 계산

공식적인 환산 공식이 공개되지 않아, **상대 비교** 방식 사용:

```
스텟 효과 점수 = (룬 효과값 / 현재 스텟) × 100 × 효과 가중치
```

**예시:**
- 현재 치명타 스텟: 6811
- 룬 효과: 치명타 +500
- 스텟 효과 점수: (500 / 6811) × 100 × 0.8 = 5.87점

### 7.3 % 효과 vs 스텟 수치 효과

| 효과 유형 | 계산 방식 |
|---|---|
| % 증가 효과 | 효과값 그대로 사용 |
| 스텟 수치 효과 | 현재 스텟 대비 증가율 계산 |

| 악사 | 솜씨 | 의지 | 연타강화 + 회복력, 궁극기 |

### 8.3 클래스별 룬 가중치 조정 (추정)

| Sub 스텟 | 관련 효과 | 가중치 보정 |
|---|---|---|
| **행운 (Sub)** | 치명타, 추가타 | +0.1 |
| **의지 (Sub)** | 강타강화, 궁극기 | 강타 +0.1 |

> **적용 방식**: 해당 클래스의 Sub 스텟과 관련된 효과에 가중치 보정 적용

### 8.4 능력치 계산 공식 (나무위키 참조)

#### 공격력 공식

```
X₁: 캐릭터의 기본 공격력 + 레벨업 카드 + 칭호
X₂: 무기 공격력 + 무기 룬 + 무기 인장
X₃: 엠블럼 + 엠블럼 인장
X₄: (무기 보너스 대미지 표기 배율 * 스텟)의 합
X₅: 목걸이 공격력 + 목걸이 인장
X₆: 펫 동행 스텟 + 패션 세트 효과 + 인챈트 등 기타 공격력 절대값 증가 항목
X₇: 공격력 % 증가의 합

공식: (X₁ + (X₂ * (1+X₃)) + X₄ + X₅ + X₆) * (1 + X₇)
```

> **핵심**: 공격력 % 증가(X₇)는 최종 곱연산으로 적용됨

#### 치명타 공식

```
조건: 지속피해를 제외한 모든 공격

K₁: 치명타 확률 % 상승의 합
K₂: 석궁사수의 거스팅 볼트 발사할 때마다 치명타 확률 75% 감소
K₃: 대부분 컨텐츠에서 권장 전투력 이상일 시 치명타 확률 30% 추가

치명타 확률 공식: (((0.5 - (1000/(치명타+2000))) + K₁) * 0.75^K₂) + K₃

K₄: 치명타 피해량 % 상승의 합

치명타 피해량 공식: (1.4 + 치명타/5000) * (1+K₄)
```

| 치명타 스텟 | 기본 확률 (K₁=0) |
|---|---|
| 2000 | 25% |
| 4000 | 33.3% |
| 6000 | 37.5% |
| 8000 | 40% |
| 10000 | 41.7% |

> **핵심**: 치명타 확률은 스텟이 높아질수록 한계효용 감소 (캡 50%)

#### 추가타 공식

```
조건: 추가타, 지속피해 등을 제외한 모든 공격

L₁: 추가타 확률 % 상승의 합

공식: (추가타/13000) + L₁
```

| 추가타 스텟 | 기본 확률 (L₁=0) |
|---|---|
| 6500 | 50% |
| 13000 | 100% |

#### 강타 강화 공식

```
조건: 대미지가 공격력의 40% 이상 [제한사항]

C₁: 강타 피해 % 증가의 합

공식: ((1+강타 강화/8500) * (1+C₁)) - 1
```

| 강타 강화 스텟 | 기본 피해 증가 (C₁=0) |
|---|---|
| 8500 | 100% (2배) |
| 17000 | 200% (3배) |

#### 연타 강화 공식

```
조건: 3초 내로 공격을 6회 이상 유지

F₁: 연타 피해 % 증가의 합

공식: ((1+ 연타 강화/8500) * (1+F₁)) - 1
```

#### 콤보 강화 공식

```
조건: 어시스트를 해제하고 공격

E₁: 콤보 피해 % 증가의 합
E₂: 1~10콤보=1, 11~50콤보=2, 51~100콤보=3, 101콤보 이상=4

공식: ((11+콤보 강화/17500) * (1+E₁) - 1) * (E₂/4)
```

#### 스킬 위력 공식

```
조건: 평타 외의 모든 공격

G₁: 자신에게 부여된 주는 스킬 피해 % 증가의 합
G₂: 자신에게 부여된 주는 피해 % 증가의 합
G₃: 자신에게 부여된 [시너지] 주는 피해 % 증가 [중복불가]
G₄: 적에게 부여된 [시너지] 받는 피해 % 증가 [중복불가]
G₅: 적에게 부여된 약화 효과: 방어구 파괴(받는 피해 10% 증가) [중복불가]

공식: (((스킬 위력/8500) * (1+G₁)) + G₂ + G₃) * (G₄ + G₅)
```

> **핵심**: 스킬 피해 증가(G₁)는 스킬 위력에 곱연산, 피해량 증가(G₂, G₃)는 덧셈

#### 광역 강화 공식

```
조건: 2개체 이상 동시 타격

I₁: 광역 피해 % 증가의 합

공식: ((1+광역 강화/8500) * (1+I₁)) - 1
```

#### 빠른 스킬 공식

```
조건: 캐스팅, 차징 스킬 사용 시

H₁: 스킬마다 부여된 캐스팅 및 차지 시간
H₂: 스킬 캐스팅 및 차지 시간 % 감소의 합
H₃: 스킬 캐스팅 및 차지 속도 % 증가의 합

스킬 준비 시간 공식: H₁ * (1 - H₂) / (1 + H₃ + 빠른 스킬)
스킬 준비 속도 공식: (1 + H₃ + 빠른 스킬) / (1 - H₂)

조건: 스킬 사용 시

H₄: 스킬마다 부여된 스킬 사용 시간
H₅: 스킬 사용 속도 % 증가의 합

스킬 사용 시간 공식: H₄ / (1 + H₅ + 빠른 스킬)
스킬 사용 속도 공식: (1 + H₅ + 빠른 스킬)
```

#### 궁극기 공식

```
O₁: 궁극기 게이지 획득량 % 증가의 합

궁극기 게이지 획득 공식: ((1 + 궁극기/7250) * (1+O₁))

조건: 궁극기 사용 시

O₂: 궁극기 대미지 % 증가의 합

궁극기 대미지 공식: ((1+ 궁극기/8750) * (1+O₂)) - 1
```

#### 무방비 (브레이크) 공식

```
조건: 브레이크 된 상대 공격 시

A₂: 무방비 피해량 % 증가
A₃: 브레이크 타입별 추가 대미지의 합

무방비 공식: (1+브레이크/5250) * (1+A₂) + A₃
```

#### 피해 감소 공식

```
조건: 피격 시

B₁: 자신에게 부여된 [시너지] 받는 피해 % 감소 [중복불가]
B₂: 자신에게 부여된 받는 피해 % 감소의 합
Y: 방어력

공식: 몬스터의 대미지 / (1+Y방어력/10000) - 피해 감소
```

#### 회복력 공식

```
조건: 회복

n₁: 회복력 % 증가의 합
n₂: 스킬 계수

공식: (회복력/8750 * (1 + n₁)) * n₂ * X공격력
```
